# Roadmap

## Completed through v0.9

- Virtual-credit wallet and immutable ledger.
- Provably-fair game engines.
- Rewards, referrals, missions, achievements and league.
- Support, notifications, TOTP and admin RBAC.
- PostgreSQL/Redis production foundation, queue, scheduler and operations dashboard.
- Live event stream, support refresh, league refresh and online presence.

## Proposed v1.0

- Versioned game-rule contracts and independent fairness verifier.
- RTP simulation and approval workflow.
- Concurrency, property and fault-injection tests.
- Full OWASP ASVS review and Content Security Policy rollout.
- Load testing and production observability.

## Later

- Optional Laravel Reverb transport.
- PWA, Vietnamese localization and accessibility review.
- Versioned API after web business rules are stable.
