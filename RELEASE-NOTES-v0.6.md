# Lucky Arcade v0.6 release notes

This release focuses on community and operations.

## Player features

- Redeem virtual-credit promo codes at `/promos`.
- Create and reply to support tickets at `/support`.
- Join the Weekly League automatically by playing; view standings at `/weekly-league`.

## Admin features

- Create and schedule promo codes at `/admin/promo-codes`.
- Answer and manage support tickets at `/admin/support`.
- Review and settle the previous Weekly League at `/admin/weekly-league`.

Weekly League rewards are fixed at 1,000 / 600 / 300 virtual credits for ranks 1–3. There is no entry fee and no cash value.
